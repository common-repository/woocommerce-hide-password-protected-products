Hide password protected products in your WooCommerce website.

== Description ==

Since WooCommerce 2.2 password protected products are not hidden by default because Catalog visibility should be set via the 'Catalog visibility' option. This plugin is intended for WooCommerce shops that want to hide all password protected products without manually editing them to set the 'Catalog visibility'.

== Installation ==

= Installing the plugin =
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for *WooCommerce Hide Password Protected Products* and click "Install now"
1. Alternatively, download the plugin and upload the contents of `woocommerce-hide-password-protected-products.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
1. Activate the plugin

== Frequently Asked Questions ==

= Are there any settings? =
No, install the plugin and all password protected products will be hidden.
